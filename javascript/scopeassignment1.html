Global variables are used to store information that we might need throughout the entire program. On the other hand, local variables are only accessible inside the function where they are defined. In today’s class, we discussed three types of functions:

  • Function Declaration: A regular function that we define normally.
  • Function Expression: Here, we assign a function to a variable.
  • Anonymous Function: A function without a name, which we usually call immediately after defining it using a pair of parentheses ().
We also talked about an important concept called Hoisting. In JavaScript, if you try to use a variable before it’s actually declared, JavaScript doesn’t throw an error. Instead, it sets the value to undefined by default. This is very different from languages like Java or C++, where trying to use an undeclared variable would cause a compilation error. Lastly, we learned about Lexical Scoping. Lexical scoping simply means that inner (nested) functions can access variables defined in their outer (parent) functions by looking up through the hierarchy.
